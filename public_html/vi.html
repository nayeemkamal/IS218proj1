Get Started with Vi

What is Vi?

Git is the most commonly used version control system. Git tracks the changes you make to files, so you have a record of what has been done, and you can revert to specific versions should you ever need to. Git also makes collaboration easier, allowing changes by multiple people to all be merged into one source.

Get started with Vi.

Step 1

Type the command "vi [filename] to create a new file

Step 2

To instert text in the file type 'i'. This opens up the instert editor where you can type in your text. For the example we types hello world.

Step 3

When done editing the file press the esc key. then to exit vi press ZZ. Check the text of the file by using the "cat [filename] command.

Vi Commands

Commands Definition
ZZ save and exit.
:q! discard all changes, since the last save, and exit.
:w save file but don't exit.
:wq save and exit.
vi [filename] Create a new file in the directory.
G move to the last line.
w move to the beginning of the next word.
nw move forward n numbber of words.
b move to the beginning of the previous word.
nb move back n number of words.
x delete a single character.
nx delete n characters
dd delete the current line
u undo the last action.
U undo all changes to the current line.
cat view a file.
less view a file. better for larger files.
ARROW KEYS move the cursor around.
^ move the cursor to beginning of current line
$ move cursor to end of the current line.
{ move backwards one paragraph.
